PKMN Project

My groupmates and I created a Pokemon battle simulator in vscode using C++. I made custom sprites, battle background, and layout for our 6 possible Pokemon. This project tested our ability to use C++ because of how complex a single turn to a Pokemon battle can be. We implemented catching and catch rate, attacking, healing, status effects, etc.

Figma App

This project was exclusively a Figma wireframe for a simple app. I completed the wireframe to learn more about the ways to utilize Figma tools. The app contains lots of placeholders
